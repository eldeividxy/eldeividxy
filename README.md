## Featured Experience

### Custom Multi-Company ERP on Odoo 18 Community

I played a key technical role in the development of a custom multi-company ERP built on **Odoo 18 Community** for a business conglomerate with a main agro-financial operation.

The system is currently in production and supports around **100 users**, **12 companies**, **6 branches**, and **6 warehouses**, centralizing business processes that were previously handled through Excel, Word templates, paper, and isolated systems.

The ERP includes **39 installable custom modules** covering agricultural credits, sales, purchases, inventory, warehouses, accounts receivable, payments, payroll, support tickets, accounting, CFDI 4.0 electronic invoicing, contracts, reports, dashboards, and multi-company security.

My work included:

* Backend development with Python and the Odoo ORM
* Custom models, business rules, computed fields, validations, workflows, wizards, and SQL constraints
* XML views, menus, actions, permissions, record rules, and access control logic
* JavaScript / OWL components and custom frontend behavior
* Full development of the CFDI 4.0 invoicing system integrated with SW Sapien PAC
* Generation of real stamped invoices, XML/PDF documents, and local test stamping tools
* Custom accounting logic and integration with internal financial movements
* Multi-company, multi-branch, and multi-warehouse permission architecture
* PDF reports, contracts, account statements, credit documents, and operational forms
* Dashboard design, CSS customization, and user interface improvements
* PostgreSQL queries, data imports/exports, data cleanup, and query optimization
* Testing, implementation, technical documentation, user training, and active support

One of the biggest impacts of the project was transforming manual and disconnected processes into automated, traceable ERP workflows. Reports that previously took days or even a full week can now be generated in minutes, contracts and credit documents can be created with one click, and mass credit charges that used to require manual work can now be processed automatically.
